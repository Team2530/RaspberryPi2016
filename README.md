# RaspberryPi2016
Contains the necessary files, and setup information for Grip on Pi

1. Obtain RaspberryPi 2 B+ with 4GB MicroSD (Class 4 or above).
2. Format MicroSD (Fat32)
3. Download https://downloads.raspberrypi.org/NOOBS_latest 
4. Follow the steps from "GRIP CORE" through "Using mjpg-streamer with a Raspberry Pi Camera" found here. Note use this link for the latest versions (which are needed): https://www.dropbox.com/sh/ixbcshu9astnfzj/AACRaa60bwZnwrTPhXUzjss3a?dl=0
